# Reno Key System

Black + Red Discord bot with control panel, key system, VIP roles, auto expiration, and webhook logging.

## Commands

### Admin:
- `!createkey <key> <duration>` — Create timed key
- `!listkeys` — Show all keys

### User:
- `!usekey <key>` — Activate key

### Buttons:
- Redeem Key
- Get Script
- Get Role
- Reset HWID
- Get Stats
